Audio Grid


This app is an html/javascript port of the great Tone Matrix tool by Andre Michelle.

If you're interested in grid/matrix based audio tools, you may want to check out some of the hardware versions available:
Monome
Tenori-On

If you have feature requests (or find a bug), please log an issue at Github. You can also leave comments on the Project Page
